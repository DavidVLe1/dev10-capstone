In order to set up and run this application, make sure to do the following steps.
1. Open MySQL and create a new connection or use an existing connection.
2. Load at least the SQL Production and example data Schema onto MySQL database and ensure the MySQL Database is Live.
3. Open the Backend through an IDE such as InteliJ, change the data inside the resources of username and password to that of your MySQL and run the app through App.js.
4. Open the frontend with an IDE such as VS Code, install the required packages through npm i.
5. Run the frontend with npm start and sign up or login with data from the SQL production schema.
